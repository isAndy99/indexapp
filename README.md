## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Run the app in a Docker container

```bash
npm run dockerize
# or
yarn start dockerize
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
